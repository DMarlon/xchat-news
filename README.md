xchat-news
==========

Plugin de RSS Feed para o xChat

#Utilizando o plugin

Para carregar o plugin na janela de conversa digite:

    /PY load *diretorio_do_arquivo*/main.py
    
Para descarregar

    /PY unload *diretorio_do_arquivo*/main.py
